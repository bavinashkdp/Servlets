All css text properties:

    property:discription

  1. color : sets the color text
  2. direction : specifies the text direction/writing direction
  3. letter-spacing : increase or decrease the space between characters in a text
  4. line-height : set the line height
  5. text-align : specifies the horizantal alignment of text
  6. text-decoration : specifies the decoration added to text
  7. text-ident : specifies the identation of the first line in a text block
  8. text-shadow : specifies the shadow effect added to text
  9. unicode-bidi : used together with the direction property to set or return wheather the text should be overridden to support multiple languages in the same document
  10. vertical-align : specifies how white-space inside an element is handled
  11. white-space : set the line height
  12. word-spacing : increase or decrease the space between words in a text
  13. line-height : set the line height
  14. line-height : set the line height
